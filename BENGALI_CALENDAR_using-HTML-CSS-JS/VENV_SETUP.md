# Virtual environment setup

PowerShell (recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Command Prompt:

```cmd
python -m venv .venv
.\.venv\Scripts\activate.bat
```

Git Bash / WSL:

```bash
python -m venv .venv
source .venv/bin/activate
```
